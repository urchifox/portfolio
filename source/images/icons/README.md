# Папка для оптимизированных векторных иконок

```shell
└── source/
    └── images/
        └── icons/
            ├── tg.svg
            ├── vk.svg
            └── yt.svg
```

Находящиеся здесь иконки станут одним спрайтом:

```shell
└── build/
    └── images/
        └── icons/                # папка для спрайта
            └── stack.svg         # спрайт
```
