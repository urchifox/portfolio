# Папка для отимизированных изображений

```shell
└── source/
    └── images/
        ├── bg.jpg
        ├── bg.webp
        ├── hero.png
        ├── hero.webp
        └── logo.svg
```

Отсюда файлы изображений при сборке без изменений попадают в `build/images/`:

```shell
└── build/
    └── images/
        ├── bg.jpg
        ├── bg.webp
        ├── hero.png
        ├── hero.webp
        └── logo.svg
```
