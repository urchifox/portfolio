# Folder for optimized images

```shell
└── source/
    └── images/
        ├── bg.jpg
        ├── bg.webp
        ├── hero.png
        ├── hero.webp
        └── logo.svg
```

During the build, image files are copied unchanged from here into `build/images/`:

```shell
└── build/
    └── images/
        ├── bg.jpg
        ├── bg.webp
        ├── hero.png
        ├── hero.webp
        └── logo.svg
```
